# 学术工程文献知识库使用指南

> 本指南为 `llm-wiki-academic-engineering` 的学术文献工作流提供详细说明和示例。

---

## 快速开始

### 1. 初始化知识库

```bash
bash install.sh --platform claude
cd ~/.claude/skills/llm-wiki  # 或你的自定义安装路径
bash scripts/init-wiki.sh "~/Documents/我的学术知识库" "船舶振动控制研究"
```

### 2. 配置研究方向

编辑生成的 `purpose.md`，明确你的研究领域和关键问题。例如：

```markdown
# 研究目的与方向

## 核心目标
系统整理船舶推进系统振动控制领域的学术文献，构建可扩展的知识网络。

## 关键问题
1. 船舶推进系统振动控制的主流方法有哪些？各自的适用边界是什么？
2. 电磁-结构耦合问题的建模方法经历了怎样的演进？
3. 有限元模态分析在工程实践中的关键参数和验证标准是什么？

## 研究范围
**涵盖：** 船舶推进轴系振动与控制、电磁-结构耦合动力学、有限元模态分析、机械系统动力学建模方法。

## 整理原则
- 优先提取可重复的实验方法、数学模型和验证数据
- 建立概念之间的双向链接网络
- 明确标注文献的置信度和方法的可重复性
- 标记未解决的工程问题
```

### 3. 配置领域词汇表

编辑 `vocabulary-academic.md`，添加你所在领域的核心术语。这些术语会被 AI 在生成知识图谱时优先使用。

```markdown
# 领域专有名词表

## 核心领域术语
- 船舶推进轴系振动
- 电磁-结构耦合
- 有限元模态分析
- 实验模态分析
- 扭转振动 / 横向振动 / 纵向振动
- 轴系对中
- 轴承动力学
- 螺旋桨激励

## 常用方法
- 传递矩阵法
- 有限元法 (FEM)
- 边界元法 (BEM)
- 多体动力学
- 子结构模态综合法
- 谐响应分析
```

## 文献输入规范

### Markdown 文献格式建议

为了获得最佳摄取效果，你的手工 Markdown 文献建议包含以下结构：

```markdown
# 文献标题

**作者**：张三, 李四
**期刊**：Journal of Ship Mechanics
**年份**：2024
**DOI**：10.xxxx/xxxxx

## 摘要
...

## 关键词
船舶振动, 推进轴系, 模态分析

## 研究方法
1. 建立了三维有限元模型
2. 采用谐响应分析方法计算振动响应
3. 通过实验验证了模型精度

## 主要结果
- 前三阶固有频率分别为 12.5 Hz, 34.2 Hz, 67.8 Hz
- 螺旋桨激励下的最大振幅发生在轴承支座处

## 结论与局限性
...

## 未来工作
建议考虑非线性轴承刚度的影响。

## 参考文献
- [1] 作者A, 标题A, 期刊A, 2022
- [2] 作者B, 标题B, 期刊B, 2023
```

## 最佳实践

1. **保持 `raw/literature/` 只读**：AI 不会修改原始文献文件
2. **定期运行健康检查**：每积累 10 篇文献后建议运行一次 lint
3. **维护 `vocabulary-academic.md`**：新出现的领域术语及时补充
4. **标注置信度**：对 AI 推断的内容主动降级为 `INFERRED` 或 `AMBIGUOUS`
5. **建立引用链接**：手动检查文献页中的 `[[引用文献]]` 是否正确链接
